# 就是因為捨不得賣，才送人

自從買了 Omnium Mini-Max，我的天天號二代 domCargo 已經擺在公園停車場風吹雨淋超過半年了。其實生了阿星後，就沒什麼心思去保養他，直接停樓下。不像以前都會搬到公寓四樓。

我第一台 domCargo 是 2017-11-24 花了一萬元，和蔡銘哲買的二手車。那時他從120mm版本升級到135mm的碟煞版本，我也就便宜接收了這台內變三速的小cargo。

他陪著我一路從新北投搬到天母，第一次大摔車受傷，在阿媽生命最後的三個月我總是騎著他往返家裡和醫院，和吳孟松一家一起單車郊遊，騎著他舉辦台北裸騎、沈默騎行，參加單車臨界量、單車通勤日、耶誕物流賽⋯⋯我的IG裡還有好多好多他的照片。

後來參與單車通勤日，為了讓站長都能用腳踏車載咖啡，所以就把天天號一代給小唐，籌組升級版的天天號二代——135mm的碟煞版本。你終究要升級的，何不一次攻頂呢！！！

天天號二代各方面都用很頂的設定，1x11的變速系統應該連牆都爬得上去，Blackburn的後貨架很帥，Brooks的草綠色皮座椅更是騷包。

二代組好沒多久，阿星出生了。我還記得當初每天往返月子中心和家裡，為了照顧貓。天天只要聽到我停天天號的聲音，就會在四樓開始喵喵叫。

等到阿星一歲了，我透過歐洲網站買了 Bobike 嬰兒座椅。第一次坐上天天號的阿星看起來好像很開心。

然而我開始沒空整天把腳踏車搬上搬下了。例行的保養也停了。把一些原本為了耍酷的零件，換上比較不需要維護的替代品。例如 Brooks 皮坐墊換成 fizik 科技坐墊。儘管如此，天天號仍然是以滿恐怖的速度劣化。永遠是橘黃色鏽斑的鏈條，破損的Bobike擋風板，斷掉的前輪鋼絲，更別提倒車讓鉤爪變形，永遠不準的變速。

一直讓他在外面吹風淋雨，其實很過意不去。雖然他現在狀況不太好，但經過懂車的人整理一下，應該也是很有用的、能載很多東西的。

曾有過一面之緣的阿桂去世了。這台出自他手筆的車，讓我不想賣掉。

上次帶阿星去改裝他的滑步車，有和VoomVoom Bikes的老闆聊了一下，他問我要不要賣他。既然這樣，就送他好了。

賣掉，感覺和阿桂的羈絆就消失了。然而送人，不知道為什麼，覺得是結交新的緣分，好像就幫阿桂延續了些什麼。我的理由毫無道理，明明在生第二胎這個需要錢的節骨眼，卻做了這個不太理性的決定。

最後幫天天號拍了幾張照片和影片，騎去VoomVoom Bikes。哎呀，沒開門。好吧，默默拿出隨身工具，把一些該拆的拆下來。隨身工具拆不了的，傳個簡訊給老闆交代一下，心裡好像放下一塊石頭。

謝謝和你一起度過的那段日子。希望你能夠再幫新的主人完成很多任務，載許多貨物，在街頭盡情奔馳！
